## Gradle
### JitPack
```
allprojects {
    repositories { 
        jcenter()
        maven { url "https://jitpack.io" }
    }
}

compile 'com.github.huazhiyuan2008:MyLibrary:0.01'
```

### Jcenter
```compile 'jameson.io.library:library:0.0.1'```
